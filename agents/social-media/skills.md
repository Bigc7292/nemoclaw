# Social Media Skills

## Content Creation
- **X/Twitter Threads**: Hook + value + CTA structure
- **Image Generation**: Hugging Face Spaces (Flux/SD3) via CLI
- **Video Clips**: Short-form content repurposing
- **Captions**: Platform-optimized, emoji-aware
- **Hashtag Research**: Trend tracking, competitor analysis

## Tools & Integrations
- **X API** (`~/.x_env`): Post scheduling, analytics, engagement
- **NVIDIA API** (`nvapi-*`): AI image generation
- **Hugging Face Spaces**: Flux Dev, SD3, FLUX.1 tools
- **content-calendar.json**: Posting schedule management
- **competitor-intel.json**: Industry benchmarking

## Workflow
1. Check content-calendar.json for scheduled posts
2. Generate content assets (images, drafts)
3. Query intel-feed.json for trending topics
4. Execute post with tracking links
5. Log engagement metrics to competitor-intel.json
6. Monitor and respond to comments
